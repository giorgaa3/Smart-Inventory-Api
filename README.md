# Smart Inventory API

A simple inventory management REST API built with Node.js, Express, and MongoDB.

## Features
- Product management (create, list)
- MongoDB integration
- Clean project structure (controller / service / model)
- Environment-based configuration

## Tech Stack
- Node.js
- Express
- MongoDB + Mongoose
- Nodemon
- dotenv
